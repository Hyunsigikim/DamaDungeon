# DamaDungeon
 스마트인재개발원 3일 미니프로젝트
 
 프로그래밍 교육 후 첫 프로젝트
 메인 언어는 JAVA
 DB는 JDBC를 이용하여 oracle가 사용되었다.
 
 클라이언트의 입력을 받아 진행되는 텍스트 기반 게임이다.
 
 
 전체적 설계
 while (게임 자체의 반복문 : 게임의 메인화면까지 들어있는 부분 )
  {
  switch (게임의 메뉴 구성단계 1. 실행 2. 랭킹 ...)
       {
       while ( 게임의 구동 부분 )
           {
           switch ( 게임 내의 메뉴 1.전투 2.채집 3.훈련 4.아이템사용 ....)
                 {}
           }
       }
  }
